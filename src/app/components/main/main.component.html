Projetado para o futuro

Apresentando um
editor extensível

Blogr possui uma interface extremamente intuitiva que permite que você se concentre em uma coisa: criar conteúdo. O editor suporta o gerenciamento de vários blogs e permite a fácil manipulação de incorporações, como imagens, vídeos e Markdown. A extensibilidade com plugins e temas fornece maneiras fáceis de adicionar funcionalidade ou alterar a aparência de um blog.

Gerenciamento robusto
de conteúdo

O gerenciamento de conteúdo flexível permite que os usuários se movam facilmente pelas postagens. Aumente a usabilidade do seu blog adicionando categorias, seções, formato ou fluxo personalizados. Com essa funcionalidade, você está no controle total.

Infraestrutura de última geração

Com confiabilidade e velocidade em mente, os data centers em todo o mundo fornecem a espinha dorsal para conectividade ultrarrápida. Isso garante que seu site seja carregado instantaneamente, não importa onde seus leitores estejam, mantendo seu site competitivo.

Livre, aberto, simples

Blogr é um aplicativo gratuito e de código aberto apoiado por uma grande comunidade de desenvolvedores úteis. Suporta recursos como realce de sintaxe de código, feeds RSS, integração de mídia social, ferramentas de comentários de terceiros, e funciona perfeitamente com o Google Analytics. A arquitetura é limpa e é relativamente fácil de aprender.

Ferramentas poderosas

Baterias incluídas. Criamos uma ferramenta CLI simples e direta que facilita a personalização e a implantação, mas capaz de produzir até mesmo os locais mais complicados.
